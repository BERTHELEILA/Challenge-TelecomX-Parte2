**TelecomX Churn Analysis**

Analiza la cancelación de clientes (churn) de TelecomX para identificar patrones que expliquen por qué los clientes se van.

**Incluye:**

Exploración y visualización de datos.

Heatmaps, boxplots y scatter plots.

Matriz de correlación de variables numéricas.

Conclusiones sobre factores que afectan la retención.

**Tecnologías:** Python, Pandas, NumPy, Matplotlib, Seaborn.

**Uso:** abrir TelecomX_2.ipynb en Jupyter Notebook o Google Colab y ejecutar las celdas.
